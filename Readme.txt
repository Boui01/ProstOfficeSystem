## รายละเอียด (Description)

- Data private user page (หน้าข้อมูลส่วนตัวสมาชิกต่างๆ)
- Transport order page (หน้าข้อมูลขนส่งปกติ)
- Transport everyday page (หน้าข้อมูลขนส่งทุกๆวัน หรือ  ทุกๆสัปดาห์)
- All transport data page (หน้าข้อมูลรวมทั้งหมดของขนส่ง)

## โค๊ด  (Code)
- โค๊ดที่สำคัญในหน้าต่างๆ จะมีอยู่ประมาณ 2 หน้าต่างหลักๆครับ  ที่จะมาให้ดูครับ (There are about 2 main windows that are important codes on various pages. There are about 2 main windows that are important codes on various pages.)

1. Transport order page (หน้าต่างขนส่ง)  
- Transport check failed page (หน้าเช็คขนส่งไม่สำเร็จ)
- Transport check successful page (หน้าเช็คขนส่งสำเร็จ)
- All transport check page (หน้าเช็คขนส่งทั้งหมด)

2. Transport every day page (หน้าต่างขนส่งทุกวัน)  
- Transport everyday check failed page (หน้าเช็คขนส่งประจำไม่สำเร็จ)
- Transport everyday check successful page (หน้าเช็คขนส่งประจำสำเร็จ)
- All transport everyday check page  (หน้าเช็คขนส่งประจำทั้งหมด)
